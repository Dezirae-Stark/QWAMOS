# Security Policy

## Reporting Vulnerabilities

Please report security vulnerabilities to: sedlhberendir@tutamail.com

**Do NOT** create public GitHub issues for security vulnerabilities.

## Disclosure Process

1. Email vulnerability details to security contact
2. Allow 90 days for fix before public disclosure
3. Credit will be given in security advisory

## Scope

In scope:
- Cryptographic vulnerabilities
- Authentication bypass
- Privilege escalation
- VM escape vulnerabilities

Out of scope:
- Physical device attacks
- Social engineering
- DoS attacks
