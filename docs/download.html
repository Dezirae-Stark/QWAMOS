About QWAMOS

QWAMOS is a mobile hypervisor OS for ARM64 devices. Current alpha builds are delivered via an Android-compatible bootstrap installer (APK + Termux tooling), which deploys the QWAMOS core and prepares the device to host multiple guest operating systems (Android, GrapheneOS, Kali, Arch, Ubuntu, etc.) inside isolated VMs.

Latest Release: v1.0.0-alpha

Released: 2024-01-01

Installation Methods

Method 1: Direct APK

# This APK is a bootstrap installer for the QWAMOS hypervisor OS.
# Enable unknown sources in Android settings
adb install qwamos-release-v1.0.0.apk

Method 2: Termux Build

pkg install git
git clone https://github.com/Dezirae-Stark/QWAMOS
cd QWAMOS
./scripts/build.sh

SHA-256 Checksums

Verify integrity before installation:

qwamos-release-v1.0.0.apk:
abc123def456789...

Verify Checksum

sha256sum qwamos-release-v1.0.0.apk

System Requirements

  • Hardware: ARM64 (aarch64) mobile SoC
  • Bootstrap Environment: Current alpha builds deploy via an Android-compatible bootstrap (Android 8.0+ firmware + Termux)
  • RAM: Minimum 4GB (8GB recommended for multiple guest VMs)
  • Storage: 10GB free storage (20GB+ recommended for multiple guest OSes)
  • Root Access: Optional but recommended for full hypervisor capabilities
  • Termux: Required for bootstrap deployment and QWAMOS core installation

Note: Future releases will support direct bare-metal installation on supported ARM64 devices.
