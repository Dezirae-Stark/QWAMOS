Installation Methods

Method 1: Direct APK

# Enable unknown sources in Android settings
adb install qwamos-release-v1.0.0.apk

Method 2: Termux Build

pkg install git
git clone https://github.com/Dezirae-Stark/QWAMOS
cd QWAMOS
./scripts/build.sh

SHA-256 Checksums

Verify integrity before installation:

qwamos-release-v1.0.0.apk:
abc123def456789...

Verify Checksum

sha256sum qwamos-release-v1.0.0.apk

System Requirements

  • Android 8.0 (API 26) or higher
  • ARM64 (aarch64) processor
  • Minimum 4GB RAM (8GB recommended)
  • 10GB free storage
  • Root access (optional but recommended)
  • Termux installed for full features
