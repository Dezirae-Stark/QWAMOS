Build Instructions

Prerequisites

# Install Termux from F-Droid
pkg update && pkg upgrade
pkg install git python nodejs-lts openjdk-17 gradle

Clone and Build

git clone https://github.com/Dezirae-Stark/QWAMOS
cd QWAMOS
./scripts/setup.sh
./scripts/build.sh

Configuration

Key configuration files:

  • /etc/qwamos/crypto.conf - Cryptography settings
  • /etc/qwamos/vm.conf - Virtual machine policies
  • /etc/qwamos/network.conf - Gateway routing rules
  • /etc/qwamos/panic.conf - Emergency wipe triggers
